 Allows users to add twitter share links to words inside a text using a shortcode. Custom text, hashtags can also be set to each post/page.

== Description ==

Allowes the users to create a link inside a text that opens the twitter share page. Custom share text and hashtags acan be added to each posts/pages using the meta box on the side on the post editor screen. Also you can set your tiwtter username using the plugins settings page, so it'll be displayed as "via <username>". You can also add an other user to promote after somebody tweeted your post.

To make a word/phrase into a link warp it around the [wp-tweet-me][/wp-tweet-me] shortcodes.

For exmaple:
`This is a [wp-tweet-me]tweet me[/wp-tweet-me] link!`

Adds a new meta box to the posts editor screen for custom share text and twitter hashtags. If there's not custom text set, it'll use the post/page title.

== Installation ==

1. Upload the`wp-tweet-me` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> WP Tweet Me settings page and set your twitter User name
4. Edit/Add a post and add custom share taxt and/or hashtags.
